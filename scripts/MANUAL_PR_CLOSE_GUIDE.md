## 操作步骤

### 1. 访问 PR 页面

点击以下链接访问对应的 PR：

- https://github.com/VisualLock/NexusAI-Manager/pull/825

### 2. 添加感谢评论

在 PR 页面底部的评论框中，粘贴以下感谢消息：

```markdown

此 PR 的更改已被手动集成到 v3.3.43 版本中。

相关更新已包含在以下文件中：
- README.md 的版本更新日志
- 贡献者列表

再次感谢您对 NexusAI Tools 项目的支持！

---

Thank you for your contribution! 🎉

The changes from this PR have been manually integrated into v3.3.43.

The updates are documented in:
- README.md changelog
- Contributors list

Thank you again for your support of the NexusAI Tools project!
```

### 3. 关闭 PR

1. 点击评论框下方的 **"Close pull request"** 按钮
2. 或者点击 **"Close with comment"** 按钮（如果你想同时添加评论）
